The xlsx and csv contain the band gap and band edge data points used for training the XGB model. The model is used to predict the band gap and band edge values of ~17000+ lead-free hybrid halide perovskites. Later they are screened to identify suitable candidates for solar water splitting application with 21 compounds satisfying all the imposed criteria. Ultimately a novel lead-free halide perovskite was validated and Moleclar dynamic simulations were carried out for the same. For more details please refer to that paper, "Prediction of Aqueous Stable Lead-Free Hybrid Halide Perovskites for Efficient Solar Water Splitting Using Machine Learning and Molecular Dynamics".(https://doi.org/10.1039/D6TC00296J)
